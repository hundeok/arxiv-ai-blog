# 전력망의 안전을 책임지는 AI 응급의사: 정형 파운데이션 모델(TFM)로 재정의하는 동적 안도도 평가

오늘 소개해 드릴 논문은 전력 계통 운영의 패러다임을 바꿀 혁신적인 연구입니다. 복잡하고 거대한 송전망의 안정성을 단 몇 줄의 데이터(In-context)만으로 실시간 진단해 내는 **정형 파운데이션 모델(Tabular Foundation Model, TFM)**의 전력 계통 적용기를 딥다이브해 봅니다.

## ✨ [3줄 핵심 요약]

1. **학습 없는 초고속 진단:** 파라미터 업데이트나 하이퍼파라미터 튜닝 없이, 오직 인컨텍스트 러닝(In-Context Learning)만으로 전력 계통의 동적 안정도(DSA)를 판별합니다.

3. **미지의 사고 예측(Generalization):** 새로 개발한 '수정된 전기적 거리 좌표(mod-EDC)'와 단 10개의 샘플만 결합해, 미지의 고장(Unseen Contingency)도 완벽히 예측해 냅니다.

## 📖 [쉽게 풀어쓴 1분 핵심]
> 💡 **1줄 요약:** 한 번도 본 적 없는 새로운 도미노 판이라도, 앞선 몇 번의 쓰러짐만 보고 전체가 쓰러질지 버텨낼지 바로 맞히는 천재 AI 감별사 이야기입니다.

전력망을 거대한 **'도미노 세트'**라고 상상해 봅시다. 

전기 발전기나 송전탑 중 하나가 고장 나는 것은 도미노 블록 하나가 쓰러지는 것과 같습니다. 이 블록이 쓰러졌을 때, 전체 도미노 판이 와르르 무너질지(불안정), 아니면 중간에 멈추고 버텨낼지(안정) 미리 알아내는 것은 매우 중요합니다. 

기존의 AI는 이 도미노 판의 결과를 맞히기 위해 **수만 번의 무너뜨리기 시뮬레이션(TDS)**을 반복해서 보며 '공부(학습)'해야만 했습니다. 게다가 도미노 판의 모양이 조금이라도 바뀌거나, 새로운 곳에서 쓰러지기 시작하면 아예 처음부터 다시 공부해야 했죠.

하지만 이 논문에서 도입한 **정형 파운데이션 모델(TFM)**은 다릅니다. 이 AI는 전 세계 수많은 정형 데이터(도미노 판)를 이미 마스터한 '천재 감별사'입니다. 새로운 도미노 판이 주어졌을 때, 시험공부를 처음부터 다시 할 필요가 없습니다. 단지 **"이 도미노 판에서는 이전에 요렇게 10번 정도 버텼거나 무너졌어"**라는 짧은 힌트(컨텍스트)만 보여주면, 그 자리에서 바로 전체 도미노 판의 운명을 정확하게 맞혀냅니다.

## 🎯 [배경 및 문제점]
> 💡 **1줄 요약:** 기존의 동적 안도도 평가(DSA) 모델들은 '1사고 1모델' 구조의 비효율성과 미지의 사고 앞에서의 무력함이라는 고질적인 한계를 가졌습니다.

### 1. 동적 안정도 평가(DSA)의 중요성
신재생 에너지(태양광, 풍력 등)의 도입률이 높아지면서 전력망의 불확실성은 전례 없이 커졌습니다. 시스템 운영자는 전력망을 경제적이면서도 안전하게 운영하기 위해, 예상치 못한 상정고장(Contingency, 예: 송전선 삼상 단락 고장 등) 발생 시 계통이 동적 평형을 유지할 수 있는지 평가하는 **동적 안정도 평가(Dynamic Security Assessment, DSA)**를 실시간으로 수행해야 합니다.

### 2. 기존 물리 시뮬레이션(TDS)의 한계
가장 확실한 방법은 미분대수방정식(DAE) 시스템을 실시간으로 푸는 **시도메인 시뮬레이션(Time-Domain Simulation, TDS)**입니다. 하지만 초대형 계통에서 TDS를 매 분, 매 초 실행하는 것은 연산량이 너무 많아 실시간 온라인 운영이 불가능합니다.

### 3. 기존 데이터 기반 ML 모델의 치명적 한계
이를 해결하기 위해 오프라인에서 TDS 데이터를 대량 생성한 뒤, 머신러닝 분류기(XGBoost, SVM, MLP 등)를 학습시켜 온라인에서 초고속 추론하는 방식이 제안되었습니다. 하지만 현업 적용에는 다음과 같은 장벽이 있었습니다.

* **모델 폭증 (Model Proliferation):** 상정 가능한 사고(Contingency) 시나리오마다 독립된 분류기 모델을 따로 학습, 튜닝 및 유지보수해야 합니다. 관리해야 할 모델이 수백, 수천 개로 늘어납니다.
* **미지 사고에 대한 일반화 실패 (Generalization Bottleneck):** 학습 데이터셋에 존재하지 않는 새로운 위치의 고장(Unseen Contingency)이나 계통 토폴로지 변화가 발생하면 예측 정확도가 바닥을 칩니다. 이는 수학적으로 통계적 분포가 바뀌는 **컨셉 시프트(Concept Shift)** 문제로, 기존 모델들이 외삽(Extrapolation)을 제대로 수행하지 못하기 때문입니다.
* **시뮬레이션 비용 병목:** 불안정한 상태(Insecure)는 극히 드물게 발생하므로, 균형 잡힌 학습 데이터를 만들기 위해 엄청난 횟수의 TDS 시뮬레이션 예산을 낭비해야 합니다.

## 💡 [핵심 기술 및 아키텍처]
> 💡 **1줄 요약:** 미세 파라미터 조정이 필요 없는 사전 학습된 'TabPFN' 모델에, 물리적 계통 특성을 담은 'mod-EDC' 인코딩을 입력해 인컨텍스트 러닝을 수행합니다.

이 논문은 모델을 매번 새로 학습시키는 패러다임을 깨고, 사전 학습된 변환기(Transformer) 기반의 **정형 파운데이션 모델(Tabular Foundation Model, TFM)**을 사용합니다. 구체적으로는 **TabPFN** 아키텍처를 채택했습니다.

### 1. Tabular Foundation Model (TabPFN v2.6)
TabPFN은 약 1억 개의 인공 합성 데이터셋을 바탕으로 사전 학습된 트랜스포머 모델입니다. 이 모델의 진가는 **인컨텍스트 러닝(In-Context Learning, ICL)**에 있습니다.

```
[지식 이전 방식의 비교]
* 기존 ML (XGBoost, SVM) : 새로운 데이터 수신 ➡️ 경사하강법(Gradient Descent) ➡️ 가중치 업데이트 ➡️ 예측
* 파운데이션 모델 (TabPFN)  : 새로운 데이터 수신 ➡️ 프롬프트/컨텍스트에 데이터 주입 ➡️ 단 한 번의 Forward Pass ➡️ 예측
```

TabPFN은 아래와 같이 사전 확률분포에 기반한 베이지안 예측 분포를 근사화합니다:

$$P(Y_{test} \mid X_{test}, X_{train}, Y_{train})$$

즉, 모델의 가중치는 완전히 고정(Frozen)되어 있으며, 오직 평가하고 싶은 소량의 $X_{train}, Y_{train}$ 데이터를 '컨텍스트(Context)'로 제공하기만 하면 초고속으로 안정도($Y_{test}$)를 추론해 냅니다.

---

### 2. 제안하는 멀티 상정고장 DSA 워크플로우

```
[오프라인 단계] 
계통 토폴로지 분석 ➡️ 소량의 대포적 시나리오 선택 ➡️ 최소한의 TDS 시뮬레이션 수행 ➡️ 컨텍스트 데이터셋 구축

[온라인 단계]
실시간 계통 상태(OC) 캡처 ➡️ 수립된 컨텍스트와 함께 단일 TFM에 쿼리(Query) 입력 ➡️ 실시간 다중 사고 안정성 레이블(TSI) 도출
```

기존처럼 사고마다 수십 개의 모델을 만들 필요 없이, **단 하나의 TFM 모델**에 서로 다른 사고 데이터셋만 컨텍스트로 갈아 끼우면 동시 진단이 가능합니다.

### 3. 핵심 기술: 수정된 전기적 거리 좌표 (mod-EDC)
미지의 상정고장(Unseen Contingency)을 예측하려면, 단순한 0과 1의 이산형 상정고장 ID가 아닌 **사고의 물리적 특성을 표현하는 연속형 공간 벡터**가 필요합니다.

기존 연구는 **전기적 거리 좌표(Electrical Distance Coordinate, EDC)**를 제안했습니다. 모선 $i$와 $j$ 사이의 임피던스 값을 이용해 전기적 거리를 정의하고, 메트릭 러닝을 통해 다차원 공간에 투영합니다.

$$d_{ij} = z_{ii} + z_{jj} - z_{ij} - z_{ji}$$

#### ⚠️ 기존 EDC의 한계
동일한 모선(Bus)에서 고장이 났으나 탈락하는 송전선이 다를 경우, 고장 해제 후의 그리드 물리적 성질(Post-fault Topology)이 완전히 달라짐에도 불구하고 기존 EDC는 동일한 값을 갖게 됩니다. 이로 인해 모델이 오작동을 일으킵니다.

####  mod-EDC (Modified EDC) 제안
연구진은 과도 안정도의 클래식 이론인 **등면적법(Equal Area Criterion, EAC)**에서 영감을 받았습니다. 가속 면적은 고장 위치에 비례하지만, 감속 면적은 사고 제거 후 그리드의 전력 동기화 능력에 좌우됩니다. 

따라서 사고 전(Pre-fault)과 사고 후(Post-fault)의 토폴로지 변화를 함께 반영하는 미분 좌표(Differential Coordinate) $Q_k$를 정의했습니다.

$$Q_k = EDC_k^{pre} - EDC_k^{post}$$

최종적으로 제안된 **mod-EDC**는 고장 모선의 위치 특성과 계통 토폴로지 변화를 모두 담은 16차원의 조밀한 물리 표상 벡터가 됩니다:

$$M_k = [EDC_k^{pre} \parallel Q_k] \in \mathbb{R}^{16}$$

## 📊 [실험 결과 분석]
> 💡 **1줄 요약:** 단 120개의 데이터만으로 학습한 TFM이 수만 개의 데이터로 학습한 기존 ML 모델들을 정밀도, 재현율, 속도 측면에서 완벽히 압도했습니다.

본 논문은 신뢰성 높은 검증을 위해 대규모 벤치마크 시스템인 **IEEE 68-Bus 시스템**에서 22개의 가혹한 상정고장 시나리오와 총 12만 개의 정상/불안정 계통 운전 상태(Operating Conditions, OCs) 데이터셋을 구축해 비교 실험을 진행했습니다.

### 1. 단일 상정고장 예측 성능 (Sample Efficiency)
다양한 학습 데이터 크기($n \in \{12, 120, 1200, 12000\}$)에서 TFM과 기존 4대 모델(XGBoost, MLP, Decision Tree, SVM-RBF)의 예측 성능을 비교했습니다.

| 모델명 | $n=12$ (Macro F1) | $n=120$ (Macro F1) | $n=12000$ (Macro F1) | 비고 |
| :--- | :---: | :---: | :---: | :--- |
| **TFM (TabPFN)** | **74.0%** | **89.5%** | **94.8%** | **초소형 데이터에서 압도적 SOTA** |
| XGBoost | 66.5% | 82.1% | 94.0% | 하이퍼파라미터 튜닝 필수 |
| SVM-RBF | 57.0% | 79.2% | 93.8% | 연산 시간 급증 |
| MLP | 학습 불가 (Overfit) | 68.0% | 92.5% | 데이터 기아 상태 취약 |

* **압도적 데이터 세이빙:** TFM은 단 **120개의 샘플**만으로 평균 **Macro F1 스코어 약 90%**에 도달했습니다. 이는 기존 모델들이 비슷한 성능을 내기 위해 수천~수만 개의 데이터가 필요했던 점과 비교하면 데이터 효율성이 **두 오더(Two Orders of Magnitude, 100배)** 개선되었음을 뜻합니다.
* **운영 안전성 극대화:** 안정도 평가에서 가장 위험한 것은 '불안정한 상태를 안전하다고 오판하는 것(False Positive)'입니다. 이를 나타내는 지표인 **특이도(Specificity)** 성능 비교 결과, TFM은 120개의 극소량 샘플 환경에서도 **81%의 특이도**를 보여주며 경쟁 모델들을 최소 10% 이상 상회했습니다.

---

### 2. 단일 TFM으로 복수 사고 동시 대처 (Multi-Contingency DSA)
단 하나의 결합 모델을 사용해 여러 사고를 동시에 다루는 멀티 라벨 SOTA 모델인 `MDSA-S`와의 비교에서도 확연한 우위를 보였습니다.

* **심각한 불균형 데이터셋 대처 능력:** 사고 발생률이 1% 미만으로 극히 극단적인 불균형을 보이는 고장 시나리오(예: 상정고장 5번)에서, 기존 `MDSA-S` 모델은 특이도가 **3.1%**로 폭락하며 사실상 예측 기능이 마비되었습니다. 반면, 단일 TFM 모델은 **61.2%의 특이도**를 유지하며 균형 잡힌 바이어스 예측 능력을 증명했습니다.

---

### 3. 미지의 상정고장(Unseen)에 대한 제로샷/퓨샷 전이 성능
연구진은 학습 중에 아예 제외했던 22번째 미지의 고장에 대해 제로샷(Zero-shot) 및 퓨샷(Few-shot) 예측 성능을 정밀 측정했습니다.

```
* Zero-shot (추가 샘플 0개): 오직 물리적 공간 투영(EDC) 정보만 활용해 추론
* Few-shot (추가 샘플 10개): 새로 정의한 mod-EDC 입력과 함께 단 10개의 신규 TDS 데이터만 제공
```

* **Zero-shot 결과:** 기존 One-Hot 인코딩이 예측 확률 50%(찍기 수준)에 머문 것과 달리, EDC 기반 입력은 추가 데이터 없이도 **81%의 Balanced Accuracy**를 기록했습니다.
* **Few-shot 결과 (TFM + 10):** 새로운 고장에 대해 TDS 시뮬레이션을 단 10회만 돌려서 컨텍스트에 찔러 넣어주는 `TFM + 10 (mod-EDC)` 기법을 적용하자, **정확도가 88%로 상승**했습니다. 이는 현실적으로 불가능한 가정(모든 타겟 레이블 정보를 사전에 안다고 가정하는 모델) 하에 구현되는 전이 학습 오라클(Oracle) 모델인 `JDA-best`와 소수점 차이로 동등한 성능 수준입니다.

## 🌍 [세상에 미치는 영향 & 실무 적용 사례]
> 💡 **1줄 요약:** 신재생 에너지 변동성으로 급변하는 전력망에 플러그앤플레이(Plug-and-Play) 방식으로 초고속 AI 보안 관제 시스템을 즉시 구축할 수 있습니다.

### 1. 전력망 운영사 (TSO/System Operators)의 물리 연산 인프라 비용 혁신
기존에는 계통의 미세한 구조적 변경이나 신재생 발전단지 연결이 발생할 때마다 수백만 달러를 들여 슈퍼컴퓨터로 TDS 시뮬레이션을 돌려 ML 모델들을 재학습시켰습니다. 
* **도입 효과:** 본 기술을 도입하면 오프라인 시뮬레이션 횟수를 무려 99% 감축할 수 있으며, GPU 한 장만으로도 대규모 송전망의 실시간 멀티 시나리오 안정도 조기 모니터링이 가능해집니다.

### 2. 가상 발전소(VPP) 및 마이크로그리드 실시간 제어 적용
수시로 계통 구조가 단절되고 병합되는 마이크로그리드 환경에서, TFM은 가중치 튜닝이 전혀 필요 없으므로 '즉시 사용 가능한(Plug-and-Play)' 실시간 과도 안정도 감시 장치로 동작할 수 있습니다.

### 3. 타 산업 도메인으로의 확장성 (개발자 및 엔지니어 관점)
본 논문의 접근법은 비단 전력 계통에만 국한되지 않습니다.
* **화학 공정 자동화 제어:** 특정 밸브 결함 시 공정 전체 파괴 안정성 예측
* **반도체 칩 설계 테스트:** 신호 선로 분기 시 발생하는 전자기적 왜곡 및 신호 무결성 판별
* **우주항공 원격 측정 감시:** 우주선 시스템 고장 시 복합 장비 영향도 실시간 판별

**이처럼 정형 파운데이션 모델(TFM)과 물리적 제약조건 임베딩(mod-EDC형)의 영리한 결합은 데이터 기아(Data Scarcity)에 허덕이는 모든 물리 엔지니어링 AI 프로젝트의 강력한 돌파구가 될 것입니다.**
