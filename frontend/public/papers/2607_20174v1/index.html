StreamHOI: 스트리밍 인간-객체 상호작용 비디오 생성을 위한 상호작용 인식 시간적 메모리 적응

> **원본 논문 정보**
> - **제목:** StreamHOI: Interaction-aware Temporal Memory Adaptation for Streaming HOI Video Generation
> - **저자:** Zejing Rao, Haoxian Zhang, Xiaoqiang Liu, Yiping Meng, Guoxin Zhang, Pengfei Wan, Fan Tang, Tong-Yee Lee
> - **발행일:** 2026-07-22
> - **PDF 링크:** [StreamHOI: Interaction-aware Temporal Memory Adaptation for Streaming HOI Video Generation](https://arxiv.org/pdf/2607.20174v1)


✨ 3줄 핵심 요약
* **실시간 상호작용 생성:** 고정된 길이의 오프라인 비디오 생성 한계를 극복하여, 지연 시간 0.75초 및 17.6 FPS의 속도로 연속적인 인간-객체 상호작용(HOI) 비디오를 생성합니다.
* **맞춤형 메모리 구조:** 트랜스포머 블록마다 상호작용 영역과 주변 영역에 대한 메모리 선호도가 다르다는 점을 활용해, 블록별로 특화된 메모리 레이아웃을 학습하는 기술을 도입했습니다.
* **장기적 일관성 확보:** 메모리 거리 스케일링 모듈을 통해 과거의 핵심 상호작용 상태를 장기적으로 참조함으로써, 긴 비디오 생성 시 발생하는 상호작용 붕괴 문제를 해결했습니다.

📖 쉽게 풀어쓴 1분 핵심
기존의 비디오 생성 AI는 짧은 영상은 잘 만들지만, 실시간으로 길게 이어지는 인간과 사물의 상호작용(예: 폰을 들고 머리를 넘기는 동작)은 중간에 동작이 무너지거나 끊기는 문제가 있었습니다. **StreamHOI**는 이를 해결하기 위해 인간과 사물의 움직임을 기억하는 방식을 최적화했습니다. 마치 사람의 뇌가 중요한 기억과 배경 기억을 다르게 처리하듯, AI 내부의 각 층(블록)이 역할에 맞는 메모리를 사용하게 함으로써, 적은 자원으로도 오랫동안 자연스럽고 정확한 상호작용을 유지하며 실시간으로 영상을 그려냅니다.

🎯 배경 및 문제점
* **오프라인 중심의 한계:** 기존 방식은 고정된 길이의 비디오만 생성 가능하여 실시간 상호작용 서비스에 부적합합니다.
* **상호작용 붕괴:** 긴 비디오를 생성하기 위해 여러 클립을 이어 붙이는 방식(Clip-chained)은 계산 비용이 높고, 시간이 지날수록 인간과 객체 간의 상호작용이 어색해지는 '상호작용 붕괴' 현상이 발생합니다.
* **스트리밍의 제약:** 스트리밍 방식은 빠른 속도를 제공하지만, 제한된 과거 정보만을 활용해야 하므로 상호작용의 일관성을 유지하기 어렵습니다.

💡 핵심 기술 및 아키텍처
* **블록별 메모리 프로파일링:** 트랜스포머의 각 블록이 무엇을 더 중요하게 기억하는지(상호작용 중심 vs 주변 환경 중심) 분석하여 블록별 특성을 파악합니다.
* **B-MST (Bias-guided Memory-Specialized Training):** 프로파일링 결과를 바탕으로 각 블록이 자신의 역할에 최적화된 메모리 레이아웃을 사용하도록 훈련합니다.
* **메모리 거리 스케일링 (Memory Distance Scaling):** 멀리 떨어진 과거의 핵심 상호작용 정보를 효과적으로 가져올 수 있도록 가중치를 조절하여 장기적인 맥락 유지력을 강화했습니다.

📊 실험 결과 분석
* **생성 성능:** 5초, 30초, 60초 단위의 테스트에서 기존의 비디오 생성 모델들보다 상호작용 정확도 및 전반적인 품질(Obj-CLIP, InternVL 지표 등)에서 우수한 성과를 보였습니다.
* **효율성:** 17.6 FPS의 속도와 0.75초의 첫 번째 청크 생성 지연 시간을 달성했습니다. 이는 대규모 생성 모델들(LC-Avatar 등)이 87~600초가 걸리는 작업을 훨씬 빠른 시간 내에 수행할 수 있음을 의미합니다.
* **유연성:** `KV Recache` 기술과 결합하여 생성 도중에 프롬프트를 실시간으로 변경해도 자연스럽게 동작이 이어지는 적응형 생성 능력을 입증했습니다.

🌍 세상에 미치는 영향 & 실무 적용 사례
* **가상 인간 및 에이전트:** 실시간으로 움직이는 가상 인간 아바타 제작이나 상호작용형 가상 에이전트 구현에 기여합니다.
* **온라인 콘텐츠 제작:** 실시간으로 상호작용이 요구되는 콘텐츠(라이브 커머스, 인터랙티브 게임 등)에서 긴 호흡의 영상을 즉각적으로 생성할 수 있습니다.
* **기술적 제약:** 다만, 사람의 상호작용이 모호한 경우 생성 품질이 다소 떨어질 수 있으며, 현재 단일 인물 위주로 훈련되어 복잡한 다인원 시나리오에서는 성능 제한이 있을 수 있음을 논문은 밝히고 있습니다.
